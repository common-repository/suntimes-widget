== Description ==

Displays sunrise and sunset times for locations in widgets


== Installation ==

This section describes how to install the plugin and get it working.

1. Go to the Plugins page in the admin area, search for `suntimes widget` and install it.
1. *Or* Extract `suntimes-widget.zip` in the `/wp-content/plugins/` directory. This will create a subfolder called `suntimes-widget` containing the plugin files.
1. Activate the `Suntimes Widget` through the 'Plugins' menu in WordPress
1. Go to `appearance->widgets`, add the widget to a widget area and then add the title, latitude and longitude. 
1. Google Maps will give you a longitude and latitude of the location you would like the sunrise and sunset times for.
